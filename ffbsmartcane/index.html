Executive Summary

Standard white canes provide tactile ground feedback but offer no warning of obstacles at knee-to-head height. Blind users must sweep repeatedly, slowing navigation and missing overhead hazards entirely.

The Force-Feedback Smart Cane integrates stereo Time-of-Flight (ToF) sensors with a 3-zone haptic handle, providing real-time directional obstacle warnings. Users receive intuitive left-center-right vibration feedback, enabling faster, safer independent navigation.

Technical Specifications

Sensor Type VL53L1X Time-of-Flight (ToF)
Sensor Quantity 2 (Stereo Configuration)
Detection Range 0.04m – 4.0m per sensor
Field of View 25° per sensor (15° toe-in)

Battery Life 6-8 hours (intermittent haptics)

Water Resistance IP54 (Splash Protected)

Build Process (2-3 Weeks)

⚠ Tools Required

▸ Soldering Station
▸ 3D Printer
▸ Multimeter
▸ Wire Strippers
▸ Heat Gun
▸ ESP32 Programmer
▸ Hot Glue Gun
▸ Screwdriver Set
PHASE 1: Sensor Integration Week 1
  • Wire VL53L1X sensors to ESP32 (I2C)
  • Set unique I2C addresses (0x54 Left, 0x29 Right)
  • 3D print sensor mounts with 15° toe-in angle
  • Mount sensors with acrylic protective windows
PHASE 2: Haptic Motors Week 1-2
  • Position 3× ERM motors (Left, Center, Right)
  • Solder to AO3400 MOSFET drivers
  • Add flyback diodes across motor terminals
  • Test PWM intensity (0-255 range)
PHASE 3: Power & Firmware Week 2
  • Wire 2× 18650 batteries in parallel
  • Connect TP4056 USB-C charging module
  • Flash ESP32 with obstacle detection firmware
  • Configure distance thresholds (50cm, 100cm, 200cm)
PHASE 4: User Testing Week 3
  • Test with blindfolded users
  • Calibrate vibration intensity
  • Verify detection range accuracy
  • Test outdoor performance
